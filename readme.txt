Para la importación de las librerías debemos seguir los pasos que aquí se muestran:


https://rukbottoland.com/blog/como-instalar-paquetes-python-con-requirementstxt/



Es obligatorio, de lo contrario no ejecutará el software.




Además, para la ejecución de los diversos archivos (main, pruebasManuales y pruebasMontadas)

debemos acceder mediante terminar a la carpeta donde se encuenta los archivos y ejecutar la siguiente línea:



python -u nombreArchivo.py

El único archivo a ejecutar para el algoritmo genético es main.py
Los archivos pruebasMontadas.py y pruebasManuales.py son apartados extra que vienen también explicados
dentro del documento.




Para cancelar o salir de los programas siempre se puede utilizar CTRL + C
